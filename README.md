                             简单学生管理系统
1.所选数据库：MySQL
2.操作环境：Windows10
3.硬件平台：Lenovo G50-70 CPU:i5-7200
4.所选开发IDE: Myeclipse2015
5.所选Web 应用服务器：tomcat7.0
6.开发语言：jsp
7.实现的功能：   
a. 管理员可以对学生、老师和课程进行增加、删除、查询。
b. 学生登录只能查看、修改自己的个人信息，查看自己所有课程成绩。
c. 班级任课教师登录后，可浏览对应课程学生基本信息和相应的成绩信息。可修改、登记相应课程的信息。
8.完成时间：2017.12.2
